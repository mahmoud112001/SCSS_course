## Description
This is the first exercise in the SCSS course, demonstrating the fundamentals of SCSS including:
- **Nesting**: Organizing selectors hierarchically
- **Variables**: Using SCSS variables for colors and other values
- **@each Loop**: Generating multiple classes dynamically
- **Mixins and Functions**: Reusable style blocks

## Project Structure
```
lab1/
├── index.html
├── README.md
└── style/
    ├── css/
    │   └── style.css
    └── scss/
        └── style.scss
```

## Features Implemented
1. **Main Section** - Styled with nesting for title, subtitle, and button
2. **Color Variables** - Primary and secondary colors using SCSS variables
3. **Button Styling** - Hover effects using nested selectors
4. **Font Size Classes** - Generated using @each loop for responsive typography

## How to Use
1. Open `index.html` in your browser
2. Modify `style/scss/style.scss` for changes
3. Compile SCSS to CSS automatically using a SCSS compiler
